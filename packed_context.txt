# Minimal Packed Context

Use only this job packet in the active prompt window:

```json
[
  {
    "job_title": "",
    "company": "",
    "required_stack": [],
    "application_url": ""
  }
]
```

Runtime rules:

- Load `core_vault/01_atomic_fact_sheet.json`.
- Load the closest baseline resume from `resumes_and_docs/categories/md/`.
- Do not load historical logs, session statuses, `core_vault/02_situational_qa_library.md`, or `core_vault/04_behavioral_star_stories.md` unless `interview_state=true`.
- Fast-path tailoring only: swap target keywords into the baseline resume.
- Apply when `stack_match >= 0.70`.
- Log to `active_application_context/job_applications_tracker.md`.
- Update `active_application_context/background_agent_state.json`.
- Terminate after `apply`, `skip`, or `blocked`.
